## 📌 Overview

SmartCart is an AI-powered grocery management and shopping assistant designed to help users manage shopping lists, monitor household inventory, track expenses, and receive intelligent product recommendations.

The platform simplifies grocery shopping through automation, budget tracking, and personalized suggestions, helping users save time, money, and effort.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Rohitj1021/Smartcart.git
cd Smartcart
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build Project

```bash
npm run build
```

### Preview Build

```bash
npm run preview
```

## 🔥 Firebase Setup

Create a `.env` file:

```env
VITE_FIREBASE_API_KEY=YOUR_KEY
VITE_FIREBASE_AUTH_DOMAIN=YOUR_DOMAIN
VITE_FIREBASE_PROJECT_ID=YOUR_PROJECT_ID
VITE_FIREBASE_STORAGE_BUCKET=YOUR_BUCKET
VITE_FIREBASE_MESSAGING_SENDER_ID=YOUR_ID
VITE_FIREBASE_APP_ID=YOUR_APP_ID
```

⚠️ Never upload `.env` files to GitHub.
